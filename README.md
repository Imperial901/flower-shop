## Run Locally

Clone the project

```bash
  git clone https://github.com/SyedaMahamFahim/Flower-For-Thought-Ecommerce-React.git
```

Go to the project directory

```bash
  cd flower-for-thought-ecommerce-react
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
